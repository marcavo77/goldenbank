# 🔑 Créer un Token GitHub - Guide Visuel

## Méthode 1 : Lien Direct

Allez directement sur cette URL (connectez-vous d'abord) :
https://github.com/settings/tokens/new

## Méthode 2 : Navigation Manuelle

1. **Cliquez sur votre photo de profil** (en haut à droite de GitHub)
2. Cliquez sur **"Settings"**
3. Dans le menu de gauche, faites défiler jusqu'à **"Developer settings"** (tout en bas)
4. Cliquez sur **"Personal access tokens"**
5. Cliquez sur **"Tokens (classic)"** (ou "Fine-grained tokens")
6. Cliquez sur **"Generate new token"** (ou "Generate new token (classic)")

## Configuration du Token

- **Note** : "NovaBank Deployment"
- **Expiration** : Choisissez une durée (90 jours, 1 an, etc.)
- **Scopes** : Cochez **`repo`** (Full control of private repositories)
- Cliquez sur **"Generate token"**
- **COPIEZ LE TOKEN IMMÉDIATEMENT** (vous ne le reverrez plus jamais !)

## Utiliser le Token

Une fois le token copié, utilisez-le comme mot de passe quand Git vous le demande.
